# AI Personal Assistant
A Python-based CLI assistant designed to help CS students manage their study workflow.

## Features
- Dynamic greetings based on time.
- Persistent To-Do list management.
- Modular architecture for adding AI API integrations.
### Update: Coding Tutor Mode
The assistant now includes a specialized 'Tutor Mode' that:
- Provides Big O notation for algorithms.
- Explains logic rather than just providing snippets.
- Supports Python, SQL, and System Design queries.
